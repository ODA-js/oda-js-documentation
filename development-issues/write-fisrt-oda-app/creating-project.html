Creating project

Create project.

Generate your new project 

yo oda-api-simple

Go to project folder (instead of <your-project-name> use the name of your project):

cd <your-project-name>

Now run project:

npm start

Open web browser. Go to localhost:3003/graphiql to check that everything is working.
